# Prompt 2: API Design Task

## Objective

Design a RESTful API for a **Task Management System** with the following requirements:

### Features Required

1. **User Management**
   - Registration
   - Authentication (JWT)
   - Profile management

2. **Task Operations**
   - Create, Read, Update, Delete (CRUD)
   - Assign tasks to users
   - Set priorities (Low, Medium, High, Urgent)
   - Add due dates

3. **Task Organization**
   - Categories/Tags
   - Search and filter
   - Sorting options

### Deliverables

Please provide:

1. **API Endpoint Design** - All endpoints with HTTP methods
2. **Request/Response Schemas** - JSON structure examples
3. **Authentication Flow** - How JWT will be implemented
4. **Error Handling** - Standard error response format
